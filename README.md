Инструкция по запуску проекта

1. Загрузка проекта

Скачайте ZIP-архив проекта с GitHub и распакуйте его в удобную директорию. Откройте распакованную папку в Visual Studio Code.

2. Установка зависимостей фронтенда

Откройте встроенную консоль Visual Studio Code с помощью сочетания клавиш: Ctrl + `
(` — это клавиша с тильдой, обычно расположена там же, где буква “ё”.)

Введите команду: npm install
Команда устанавливает все зависимости, указанные в файле package.json, включая библиотеки, необходимые для работы фронтенда.

3. Установка зависимостей бэкенда

В той же консоли (или открыв новую) установите зависимости Django: pip install -r requirements.txt
Команда устанавливает все пакеты, необходимые для работы бэкенда на Django.

4. Запуск проекта

Для запуска фронтенда и бэкенда одновременно выполните команду: npm run dev
После этого запускаются два сервера:

Фронтенд:
http://localhost:3000

Бэкенд (Django):
http://localhost:8000

Админ-панель Django:
http://localhost:8000/admin

5. Описание проекта

Проект представляет собой сайт по продаже одежды. Пользователь может просматривать товары, добавлять их в корзину, оформлять заказы и указывать данные для доставки и оплаты.
Администратор в административной панели Django может видеть всю информацию о каждом заказе: товары, которые заказал пользователь, его фамилию, имя, отчество, телефон, адрес доставки, способ доставки и способ оплаты.
